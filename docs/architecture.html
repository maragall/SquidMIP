SquidHCS — Architecture

Squid acquisition → max-projected plate. One FOV per well, no stitching. draft v0.1
CPU · I/O-bound · no GPU. Wells processed in parallel by a worker pool; only one well's z-stack is resident per worker, so memory is bounded regardless of plate size (1536wp OK). Bottleneck is TIFF decode/disk, not compute.
